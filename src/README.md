# Civico Webapp

## Lokal starten

1. Node.js 20+ installieren
2. Im Projektordner ausführen:

```bash
npm install
npm run dev
```

3. Im Browser die angezeigte lokale URL öffnen.

## Build

```bash
npm run build
npm run preview
```

## Enthalten

- Freiwillige
- Vereine / Organisationen
- Gemeinde-Dashboard
- Admin-Dashboard
- Login / Registrierung
- Rechtliche Seiten

## Hinweise

- Die App nutzt Supabase.
- Gemeinde/Admin sind als Frontend-Struktur vorbereitet und müssen bei Bedarf noch mit den finalen Tabellen und Rollen in Supabase verbunden werden.
